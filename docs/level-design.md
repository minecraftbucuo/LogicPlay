# LogicPlay 关卡设计

这个文件用于逐关设计正式关卡。当前先只记录第一关，后续关卡确认后再继续追加。

## Level 1：基础移动

- 状态：✅ 已实现
- 关卡目标：让玩家学习最基础的移动。
- 教学重点：使用 `robot.move_forward()` 控制机器人向前移动。
- 通关目标：移动到终点即可通关，并解锁下一关。
- 地图尺寸：5x5。
- 起点：`{ x: 0, y: 2, direction: 'right' }`
- 终点：`{ x: 4, y: 2 }`
- 障碍物：无。
- 收集品：无。
- 特殊机制：进入第一关时先显示一段故事背景，居中展示在屏幕中间；玩家点击确认后再开始游戏。
- 通关条件：机器人到达终点格。
- 初始代码：提供最基础的一行移动代码，让玩家理解调用方式。

```python
robot.move_forward()
```

- 设计说明：这一关只让玩家建立“写一行代码，机器人移动一步”的直觉，不引入转向、障碍物或其他机制。

### 第一关开场故事

- 展示时机：玩家进入第一关后，游戏正式开始前。
- 展示方式：屏幕中央弹窗，背景轻微变暗或模糊。
- 交互方式：玩家点击“开始任务”后关闭弹窗，再进入可操作状态。
- 故事文案：欢迎来到 LogicPlay。你的机器人已经在训练场待命，但它还不会自己行动。现在，请写下最基础的移动指令，让机器人一步一步抵达绿色终点。
