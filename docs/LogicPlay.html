01创意介绍

把编程概念变成可玩的游戏关卡，在闯关中自然掌握编程逻辑

OVERVIEW
🎮

LogicPlay 是什么？

LogicPlay 是一款纯前端的编程练习游戏。玩家在浏览器中编写 Python 代码，控制机器人在网格地图上移动、转向、感知环境，逐关完成挑战。关卡从基础移动开始，循序渐进地引入 for 循环、if 条件判断、函数封装，直到迷宫策略。每一关对应一个编程知识点，配合开场故事和提示说明，让玩家在解决具体问题的过程中建立"代码控制行为"的直觉。

"编程入门的最大障碍不是学不会，而是看不到 —— LogicPlay 让每一行代码都有即时反馈。"
CORE
🎲

神秘方块机制

每次进入关卡随机生成通路，迫使玩家使用条件判断而非背诵固定路线，确保真正理解概念。

TECH
🌐

纯前端零后端

基于 Pyodide 在浏览器内直接运行 Python，无需安装任何环境，打开链接即可游玩。

DESIGN
📖

叙事驱动关卡

每关配有开场故事、提示说明和标准答案。将编程概念融入关卡叙事，学习不再枯燥。

AI
🤖

AI 协作开发

项目全程使用 AI 辅助开发：快速生成数据结构与引擎代码，协助方案取舍与调试，显著提升开发效率。

03价值意义

让编程学习从"看不到"变成"玩得会"

01

降低入门门槛

无需安装环境，打开浏览器就能写 Python 并看到效果，消除"配环境"这道最常见的劝退关卡。

通过随机神秘方块、多地图迷宫等机制，确保玩家必须理解概念本身才能通关，而非记忆固定答案。

每一关有故事背景、明确目标、即时反馈，把"刷题"变成"闯关"，让学习过程充满成就感。
