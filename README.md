# LogicPlay

用 Python 控制机器人闯关的编程练习游戏。

## 在线体验

<https://minecraftbucuo.github.io/LogicPlay/>

## 作品说明

**作品名称**：LogicPlay —— 用 Python 控制机器人闯关的编程练习游戏

**核心痛点**：编程初学者在学习循环、条件判断、函数等基础概念时，往往缺乏直观的练习场景，传统刷题方式枯燥且难以建立"代码控制行为"的直觉。LogicPlay 通过可视化的机器人闯关玩法，让学习者在解决具体问题的过程中自然掌握编程逻辑。

**核心玩法与功能亮点**：
- 玩家在浏览器中编写 Python 代码，控制机器人在网格地图上移动、转向、感知环境，逐关完成挑战。
- 关卡循序渐进：从基础移动、绕过障碍，到 for 循环、if 条件判断、函数封装，直至迷宫策略，共 6 关。
- 创新机制：神秘方块（每次随机生成通路，迫使玩家使用条件判断而非固定路线）、多测试用例迷宫（验证通用策略而非硬编码路径）。
- 基于 Pyodide 在浏览器内直接运行 Python，无需后端，打开即玩。
- 每关配有开场故事、提示说明和标准答案，支持代码执行速度调节和音效反馈。

**创意设计**：将编程概念融入关卡叙事，每一关对应一个编程知识点，通过游戏机制（如随机神秘方块、多地图迷宫）确保玩家必须理解概念本身才能通关，而非记忆固定答案。

**AI 协作开发**：项目全程使用 AI 辅助开发。AI 在以下环节发挥了核心价值：快速生成关卡数据结构与引擎代码，大幅缩短原型搭建周期；在功能迭代中协助方案取舍，例如神秘方块的随机机制设计、多测试用例的关卡架构决策；辅助调试 Pyodide 沙箱与游戏引擎的异步步进逻辑。AI 的核心价值在于将设计意图快速转化为可运行代码，并在多种实现方案中提供权衡分析，显著提升开发效率。

## 技术栈

- React + TypeScript + Vite
- Pyodide（浏览器内运行 Python）
- Web Audio API（音效合成）
